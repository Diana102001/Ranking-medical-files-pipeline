# Ranking medical files pipeline 
Developed an NLP pipeline for ranking medical records using preprocessing, classical methods, transformers, and LLMs to improve query-based retrieval accuracy.

## Key features
- Text preprocessing (cleaning, tokenization, normalization)
- Classical NLP methods (TF-IDF, similarity scoring)
- Transformer-based models for semantic understanding
- Integration of Large Language Models (LLMs)
- Improved query-based retrieval for high search accuracy
